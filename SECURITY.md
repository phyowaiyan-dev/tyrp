# Security Policy

If you find a security issue in TYRP, please report it privately instead of opening a public issue.

## What to report

- Vulnerabilities in the CLI or package behavior
- Dependency issues that could affect users
- Sensitive data exposure

## What to include

- A clear description of the issue
- Steps to reproduce
- Any relevant logs, screenshots, or proof of concept

If the issue is urgent, mention the severity and expected impact.
