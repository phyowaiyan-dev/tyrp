# Code of Conduct

This project follows a simple standard: be respectful, be constructive, and assume good intent.

## Expected behavior

- Use welcoming and inclusive language.
- Be respectful of different viewpoints and experience levels.
- Accept constructive feedback gracefully.
- Focus on what is best for the project and the community.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language.
- Trolling, insults, or deliberately disruptive behavior.
- Publishing private information without permission.

If you see a problem, please report it privately to the project maintainer.
